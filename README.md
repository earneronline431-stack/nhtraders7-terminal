# NHTRADERS7 Market Intelligence Terminal

## Railway pe Deploy Karna

### Step 1 — GitHub pe upload karo
1. github.com pe jao
2. New repository banao: `nhtraders7-terminal`
3. Yeh 3 files upload karo:
   - `package.json`
   - `server.js`
   - `public/index.html`

### Step 2 — Railway pe deploy karo
1. railway.app pe jao
2. "Start a New Project" click karo
3. "Deploy from GitHub repo" select karo
4. Apna `nhtraders7-terminal` repo select karo
5. Railway automatically detect karega Node.js
6. Deploy ho jayega — URL milegi jaise:
   `https://nhtraders7-terminal.up.railway.app`

### Step 3 — Done!
- Terminal live ho jayega
- TradingView charts real-time dikhenge
- Sidebar mein live prices update honge
